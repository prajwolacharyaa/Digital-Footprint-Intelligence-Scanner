# Installation Guide

---

# Windows Setup

## 1. Install Python

Download Python:

https://www.python.org/downloads/

Verify:

```bash
python --version
```

---

## 2. Clone Repository

```bash
git clone https://github.com/prajwolacharyaa/Digital-Footprint-Intelligence-Scanner.git
```

---

## 3. Enter Project

```bash
cd Digital-Footprint-Intelligence-Scanner
```

---

## 4. Create Virtual Environment

```bash
python -m venv venv
```

---

## 5. Activate Environment

```bash
venv\\Scripts\\activate
```

---

## 6. Install Requirements

```bash
pip install -r requirements.txt
```

---

## 7. Run Flask App

```bash
python app.py
```

---

## 8. Open Browser

```text
http://127.0.0.1:5000
```

---
